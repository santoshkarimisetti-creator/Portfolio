AI‑Powered Driver Safety & Passenger Security System

FlaskReact.jsMongoDBML Inference

A multi-component real-time safety system built from scratch. An Android app captures the live camera feed, a Flask backend runs ML inference to detect driver drowsiness and distraction, and a React.js dashboard displays live status and alert history — all backed by MongoDB. The hardest engineering challenge was keeping latency low across the full chain; solved by decoupling the ML inference step from the alert dispatch so neither blocks the other.
